# AI Resume Analyser

AI Resume Analyser is a React, Tailwind CSS, and Puter.js based web application that allows users to upload resumes, receive ATS scoring, and get AI-driven feedback for job applications. The app features client-side authentication, secure resume storage, and a clean, responsive UI built with reusable components.

## Features

- Client-side authentication without a backend using Puter.js  
- Upload and securely store resumes in the cloud  
- AI-powered resume-job matching and ATS scoring  
- Modern, responsive UI built with React and Tailwind CSS  
- Modular and maintainable TypeScript codebase  

## Tech Stack

- React  
- Tailwind CSS  
- TypeScript  
- Puter.js (serverless auth, storage, AI)  
- React Router v7  
- Vite  
- Zustand (state management)  

## Getting Started

### Prerequisites

- Node.js and npm installed  
- Git installed
git clone https://github.com/RajvardhanDhakar/AI-Resume-Analyser.git
cd AI-Resume-Analyser
npm install
npm run dev


Open [http://localhost:5173](http://localhost:5173) in your browser to use the app locally.
